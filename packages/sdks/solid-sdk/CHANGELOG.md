# Changelog

## [1.0.0] - 2024-01-07

### Added
- Initial release of @descope/solid-sdk
- DescopeProvider component for Solid applications
- useDescope, useSession, and useUser hooks
- Descope flow component
- Widget components (UserManagement, RoleManagement, AccessKeyManagement, AuditManagement, UserProfile, ApplicationsPortal, TenantProfile, OutboundApplications)
- Full SolidJS signals-based reactivity
- SSR/SolidStart support (client-side only)
